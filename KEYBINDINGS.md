# Keybindings Quick Reference

## i3 Window Manager

### Basic
- `Mod+Enter` - Open terminal (Kitty)
- `Mod+d` - Open application launcher (Rofi)
- `Mod+Shift+q` - Close focused window

### Window Focus
- `Mod+j/k/l/;` or `Mod+Arrow Keys` - Change focus between windows

### Window Movement
- `Mod+Shift+j/k/l/;` or `Mod+Shift+Arrow Keys` - Move focused window

### Layouts
- `Mod+h` - Split horizontal
- `Mod+v` - Split vertical
- `Mod+s` - Stacking layout
- `Mod+w` - Tabbed layout
- `Mod+e` - Toggle split layout
- `Mod+f` - Toggle fullscreen

### Workspaces
- `Mod+1-0` - Switch to workspace 1-10
- `Mod+Shift+1-0` - Move window to workspace 1-10

### System
- `Mod+Shift+c` - Reload configuration
- `Mod+Shift+r` - Restart i3
- `Mod+Shift+e` - Exit i3
- `Mod+r` - Resize mode

## Neovim

### Normal Mode
- `Space` - Leader key
- `Space+w` - Save file
- `Space+q` - Quit
- `Ctrl+h/j/k/l` - Navigate between splits

### Window Resizing
- `Ctrl+Up/Down/Left/Right` - Resize windows

### Buffer Navigation
- `Shift+h` - Previous buffer
- `Shift+l` - Next buffer

### Visual Mode
- `<` / `>` - Indent/unindent (stays in visual mode)

## Kitty Terminal

- `Ctrl+Shift+c` - Copy to clipboard
- `Ctrl+Shift+v` - Paste from clipboard

## Tips

1. Customize keybindings in respective config files
2. The Mod key is set to Super (Windows key) by default
3. Check individual config files for more keybindings and options
